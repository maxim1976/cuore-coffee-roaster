# VIP Annual Membership Section — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `#vip-plan` section between the beans catalog and contact sections — dark premium style, bilingual zh-TW/EN with JA/KO i18n, featuring a split two-column layout with price card on left and benefits checklist on right.

**Architecture:** Three files are touched in order: `i18n.js` (translations), `index.html` (nav link + section markup), `style.css` (VIP styles). No JS logic is needed — the existing `data-i18n` system handles all language switching automatically once keys are added.

**Tech Stack:** Vanilla HTML, CSS custom properties, existing `data-i18n` i18n system. Dev server: `npm start` (runs `serve public`).

> **Note on testing:** This is a static site with no test framework. Each task ends with a browser verification step in place of automated tests.
