# 科雷精品咖啡館 Cuore Coffee Roaster

專業精品咖啡網站 - 花蓮自家烘焙咖啡館

## 網站特色

- **雙語菜單展示** - 中英文完整呈現
- **響應式設計** - 完美支援手機、平板、桌機
- **現代化設計** - 暖木色調搭配工業風格
- **流暢動畫** - 滾動顯示動畫效果
- **優化 SEO** - 語意化 HTML 結構
